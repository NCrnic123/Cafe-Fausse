
The purpose of this file is to explain how AI tools were used throughout the creation of this website. AI played a major role in many stages of the project.

The initial version of the website was generated by uploading the requirements document for the Café Fausse website into Codex. Codex did an impressive job producing all the core files needed to get the site up and running. However, I quickly learned that Codex isn’t always perfect on the first attempt. That’s where GitHub Copilot and Copilot in VS Code became extremely useful. I relied on a combination of AI tools — Copilot in VS Code, Copilot on GitHub, and Codex/ChatGPT — to debug issues, refine the code, and improve functionality.

Using the base code from Codex, along with the training I received in this program, I added the images I created with ChatGPT, updated the wording, adjusted the colors, and made several cosmetic improvements to match the vision for the site.

The most challenging part of the build was getting the Newsletter Sign‑Up and Reservations features working correctly. These required a lot of trial and error, and although the AI tools helped, they often needed me to first identify the underlying problem before they could generate the correct solution. Codex was especially effective once I knew exactly what I wanted and had a general idea of how the code should work.

Overall, I was amazed by what these AI tools can do. Using AI extensively to create websites, apps, and more isn’t just the future — it’s already the present.
